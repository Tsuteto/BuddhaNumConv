# 仏典数詞変換
指数表記から複雑な上数の位取りを用いる仏典数詞に変換するネタスクリプト。  
せっかく巨大数が扱える命数法があるのに使わないのはもったないと思って作りました。  
実用性は定かではありませんけど、無量大数を超える数が実際に表記できるようになる程度の価値はあると思います。。

## Dependencies
- BigNumber.js
- jQuery
- FontAwesome
